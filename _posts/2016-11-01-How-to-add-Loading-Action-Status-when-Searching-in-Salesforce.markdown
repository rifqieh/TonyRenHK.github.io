## Question

How to add 'Loading' Action Status when Searching in Salesforce?






## Demo


How to add'Loading' when searching?
The following code can be added in Visualforce page.
javascript code:

```
$(document).on('mousemove', function(e){
            $('#loadtext').css({
               left:  e.pageX,
               top:   e.pageY
            });
        });
```
